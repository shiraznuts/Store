# Store — Shiraz Nuts & Sweets

فروشگاه آنلاین Shiraz Nuts & Sweets — Firebase/Firestore + Cloudinary، هاست‌شده روی GitHub Pages.

## فایل‌ها

- **index.html** — سایت مشتری (عربی + انگلیسی با دکمه تعویض زبان). محصولات، دسته‌بندی‌ها و سفارش‌ها مستقیم از Firestore خونده می‌شن.
- **admin.html** — پنل مدیریت (محصولات، دسته‌بندی‌ها، بنرها، تخفیف‌ها، کوپن‌ها، سفارش‌ها، کاربران). ورود با ایمیل مدیر.
- **bulk-import.html** — ابزار یک‌باره برای وارد کردن گروهی محصولات به Firestore:
  - دکمه «شروع وارد کردن محصولات»: کاتالوگ اولیه‌ی ۴۷ محصول (فقط یک‌بار اجرا کن — اجرای دوباره باعث تکراری‌شدن محصولات می‌شه).
  - دکمه «وارد کردن ۱۷ محصول چای جدید»: محصولات چای/دمنوش تأمین‌کننده‌ی Heidarpour Investment LLC، جدا از کاتالوگ اصلی (اجرای دوباره‌ش هم باعث تکرار می‌شه — فقط یک‌بار بزن).
  - دکمه «حذف همه محصولات»: همه‌ی محصولات Firestore رو پاک می‌کنه — با احتیاط استفاده کن.
- **products.json** — فایل قدیمی و **استفاده‌نشده** توسط سایت فعلی (نه index.html و نه admin.html بهش رفرنس نمی‌دن، چون هر دو مستقیم با Firestore کار می‌کنن). فقط برای آرشیو نگه داشته شده.

## اضافه کردن عکس محصولات جدید

بعد از زدن دکمه‌ی وارد کردن محصولات (چه ۴۷ تای اصلی، چه ۱۷ تا چای)، عکس‌ها خالی می‌مونن. برای هر محصول از پنل مدیریت (admin.html) وارد بشو و عکسش رو آپلود کن (میره روی Cloudinary).
